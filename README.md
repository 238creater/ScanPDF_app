# Scan to PDF

スマホで紙を撮影して、スキャン風に補正したPDFを作る静的Webアプリです。

## Features

- カメラ撮影
- 画像ファイルから追加
- 紙の四隅の自動推定
- 撮影後の四隅ドラッグ調整
- 辺の中央ハンドルによる端の微調整
- 台形補正
- スキャン / カラー / グレー / 白黒フィルター
- A4 / Letter / 画像サイズのPDF出力
- 複数ページの並べ替え、削除、再調整
- iPhone向けの共有ボタン
- ブラウザ内履歴
- デフォルト設定の保存
- ホーム画面下固定の撮影開始 / インポート
- 写真アプリ風の撮影専用画面
- 撮影中の紙輪郭プレビュー
- フラッシュ選択UI
- 補正後に「追加して撮影」または「追加画面へ」を選択

## Usage

`index.html` をブラウザで開くか、GitHub Pagesで公開して使います。

ホーム画面下部の「撮影開始」から撮影専用画面へ移動します。撮影画面では紙の輪郭を自動検出して表示します。撮影後は補正画面で四隅や辺を指で調整し、仕上げや明るさを整えてから「追加して撮影」または「追加画面へ」を選びます。

追加画面ではページの並び替え、削除、再補正、PDF保存、JPEG保存、共有ができます。

## GitHub Pages

このリポジトリをGitHubにpushしたあと、Repository settingsの Pages から公開できます。

おすすめ設定:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

公開URLは `https://<user>.github.io/<repo>/` の形式になります。

## iPhone Notes

iPhoneでカメラを使うにはHTTPS配信が必要です。GitHub PagesはHTTPSなので、そのままカメラ撮影を使えます。

共有ボタンはSafariのWeb Share APIを使います。共有先から「ファイルに保存」、AirDrop、メッセージ送信などを選べます。共有が使えない環境では「PDF保存」を使ってください。

フラッシュはブラウザと端末が `torch` 制御に対応している場合のみ実際に切り替わります。非対応環境では選択UIだけが表示されます。

## Privacy

画像処理とPDF生成はブラウザ内で行います。サーバーへ画像を送信しません。

履歴はブラウザの `localStorage` に保存されます。端末やブラウザを変えると履歴は共有されません。

## Files

- `index.html`: 画面構造
- `styles.css`: レイアウトと見た目
- `app.js`: カメラ、画像補正、履歴、PDF生成
